# Woodpecker Method

This project makes it easier to apply the Woodpecker Method: go through a high volume of positions repeatedly to increase your chess skills.
The puzzles in this app are based on second book and focus on positional puzzles. Note that only the first 296 puzzles are included.

## Firebase

This app requires an account to login (only Google account is supported). This is used to store which puzzle you should solve next. No other information is stored.
